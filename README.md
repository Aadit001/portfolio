# Aditya Tiwari — Product Manager Portfolio

A clean, responsive single-page portfolio showcasing 8+ years of product management experience across platform engineering, payments, AI, and telecom.

## Live Site

🔗 [aadit001.github.io/portfolio](https://aadit001.github.io/portfolio)

## Sections

- **Hero** — Introduction and CTA
- **About** — Background and key stats
- **Achievements** — Platform scale, conversion growth, technical strategy
- **Projects** — TechPulse, AI-Driven Cloud Resource Scheduling (M.Tech Thesis)
- **Skills** — Product Management, Technical, Tools, AI & ML, Methodologies

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Inter)
- Hosted on GitHub Pages

## Setup

Clone and open `index.html` in a browser — no build step required.

```bash
git clone https://github.com/Aadit001/portfolio.git
cd portfolio
open index.html
```
